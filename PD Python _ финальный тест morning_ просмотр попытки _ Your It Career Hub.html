Что такое интерпретатор в Python

Вопрос 1 Ответ
a.

Интерпретатор в Python - это среда разработки, предоставляющая пользователю удобный интерфейс для написания, тестирования и отладки программ на языке Python.

b.

Интерпретатор в Python - это программа, которая выполняет исходный код на языке Python, переводя его в машинный код непосредственно во время выполнения.

c.

Интерпретатор в Python - это инструмент для статической проверки синтаксиса кода на предмет ошибок до его компиляции в исполняемый файл.

d.

Интерпретатор в Python - это библиотека, позволяющая работать с графическими интерфейсами и создавать визуальные приложения на языке Python.

Отзыв

Ваш ответ верный.

Интерпретатор в Python - это программа, которая выполняет исходный код на языке Python, переводя его в машинный код непосредственно во время выполнения.
